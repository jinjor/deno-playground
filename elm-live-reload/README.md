# Elm Live Reload

```
deno https://raw.githubusercontent.com/jinjor/deno-playground/master/elm-live-reload/elm-live-reload.ts src/Main.elm src/index.html --allow-net --allow-run
```

## install

Requirements:

- [denoget](https://github.com/syumai/deno-libs/tree/master/denoget).

```
denoget https://raw.githubusercontent.com/jinjor/deno-playground/master/elm-live-reload/elm-live-reload.ts
elm-live-reload src/Main.elm src/index.html
```
